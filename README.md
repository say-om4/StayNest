# PG Life

A modern Full Stack PG Booking Web Application built using React, PHP, MySQL and Bootstrap.

## Features

* User Registration & Login
* Secure Password Hashing
* Search PG by Name & City
* Filter by Room Type
* View PG Details
* Wishlist
* Book PG
* Admin Dashboard
* Add PG
* Edit PG
* Delete PG
* Image Upload
* Responsive Design
* About & Contact Pages

## Tech Stack

### Frontend

* React.js
* Bootstrap 5
* React Router
* Axios
* React Toastify

### Backend

* PHP
* MySQL

## ⚙ Installation

### 1. Clone Repository

git clone <repository-url>

### 2. Database

Import:

database/pglife.sql

using phpMyAdmin.

### 3. Backend

Copy backend folder inside htdocs.

Example:

xampp/htdocs/PG-Life/backend

### 4. Frontend

Install dependencies:

npm install

Run:

npm run dev
